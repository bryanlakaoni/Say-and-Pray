## üßê What is this?
* Users often forget prayer schedules (even the prayer itself)
* This app provides offline access to local prayer list
* This app also provides online access to liturgical calendar
* This app has a built in API to let user have their daily Scripture (Bible verse)

* ‚úÖ **Offline Mode:** Access prayers without internet.

## üöÄ How to Run Locally

### Prerequisites
* **Android Studio** (Latest version recommended)
* **Java JDK 17** (or compatible version)

### Steps
1. **Clone the repository**
   ```bash
   git clone [https://github.com/bryanlakaoni/Say-and-Pray.git](https://github.com/bryanlakaoni/Say-and-Pray.git)
2. **Open in Android Studio**
   * Launch Android Studio.
   * Click "Open" and select the cloned project folder.
4. **Sync & Build**
   * Wait for Gradle to finish syncing (bottom right status bar).
   * If prompted, accept to install any missing SDK platforms.
5. **Run the App**
   * Connect an Android device via USB OR create a Virtual Device (Emulator).
   * Click the green Run 'app' (‚ñ∂) button in the toolbar.
